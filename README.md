# stove-guy
Tavus hackathon project for conversational cooking

## Virtual presentation camera (`camera/`)

A localhost web app + OBS-backed virtual camera that feeds an ordered set of
images into a selectable macOS camera, driven like a slide deck — each slide
either auto-advances after N seconds or waits for a manual press. Includes an
editor (upload/reorder/timing) and a presentation mode with a live cursor.

See [`camera/README.md`](camera/README.md) for setup and usage.
